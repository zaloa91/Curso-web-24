Atributos en elementos HTML

Todos los elementos pueden contener atributos. Los atributos proveen información adicional.

Los atributos se especifican en la etiqueta de apertura. Se componen de dos partes, nombre y valor.

name="value"

Href

Es el atributo básico de los links (a). Las (a) define un hipervínculo, el atributo href define la URL de la página donde nos va a llevar el hipervínculo. Página de Búsqueda.

Src

La etiqueta (img) se usa para insertar una imagen en una pagina HTML. El atributo src indica el camino o dirección de la imagen a mostrar.

Width y height

Los atributos width, ancho y height, altura definen el tamaño que queremos otorgar a la imagen de nuestra página.

Alt

El atributo alt define el texto alternativo para una imagen. Este texto se mostrara si por alguna razon la imagen no puede cargarse.

Style

Este atributo se usa para añadir estilos visuales a cualquier elemento. Como por ejemplo, color, fuente-font, tamaño , size etc.

Clic aqui para mas detalles sobre style

Title

Este atributo se usa para proveer info extra sobre un atributo.

Esta informacion se muestra en forma de tooltip (cuadrito con informacion) al dejar el cursor sobre el elemento.
