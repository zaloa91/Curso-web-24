Selectores CSS

Selectores se usan para "encontrar" los elementos HTML a los que queremos otorgar estilo.

Tiene 5 categorias:

  • Selectores simples: nombre, clase, id.
  • Selectores combinadores: Seleccionan elementos basandose en una relación especifica entre ellos.
  • Selectores de pseudo-clase: Seleccionan elementos basandose en un estado especifico.
  • Selectores de pseudo-elementos: Seleccionan y estilizan ciertas partes de un elemento.
  • Selectores de atributo: Seleccionan elementos basandose en un atributo o en el valor de un atributo.

Selector ID:

Selecciona un único elemento basándose en el atributo ID.

El atributo ID es unico por cada elemento. Si en el documento hay dos elementos con el mismo ID, está se aplicara solamente al último (orden de lectura).

Selectores combinadores:

Un combinador es un selector que explica la relación entre dos elementos.

Hay cuatro tipos distintos de combinadores:

  • Selector desecendiente (espacio)
  • Selector child/hijo (>)
  • Selector adyacente/hermano (+)
  • Selector de hermano general (~)

Selector descendiente:

Se usa para seleccionar a todos los descendientes de un elemento especificado.
