Pseudo-clases de CSS

Una pseudo-clase se usa para definir un estado especial de un elemento.

Ejemplos:

  • Estilizar un elemetento cuando pasamos el ratón por encima de este.
  • Estilizar los links visitados y sin visitar de forma diferente.
  • Estilizar un element cuando obtiene el foco.

Diferentes estados de los links:

Esto es un link

Un link tiene 4 estados distintos: link (por efecto, sin visitar), hover (cuando pasamos el cursor por encima) y active (en el momentoen el que estamos clicando sobre el link.)

a:hover debe ir después de a:link y a:visited para poder ser efectivo.

a:active debe ir después de a:hover para poder ser efectivo.
